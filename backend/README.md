# Backend demo

Server Express minim pentru:
- health check
- leaderboard demo
- save/load profile local
- normalizare reward pentru bridge result

## Start
```bash
npm install
npm run dev
```

## Endpoints
- `GET /health`
- `GET /leaderboard`
- `POST /save/:profileId`
- `GET /save/:profileId`
- `POST /iframe-result`
